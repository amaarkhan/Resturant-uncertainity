# Phase 4 Go-To-Market Validation Plan

## Purpose
Test whether a repeatable acquisition and conversion path exists for the MVP.

## Channels to Test
1. Direct outreach to small restaurant owners.
2. Local market flyers with QR.
3. Social media targeted messaging for cloud kitchens.

## Experiments
1. Onboarding friction test
- Measure setup completion rate.
- Goal: identify drop-off points.

2. Value messaging test
- Compare 2-3 value proposition variants.
- Goal: improve activation and retention.

3. Pricing signal test
- Collect willingness-to-pay bands.
- Goal: estimate feasible pricing floor/ceiling.

## GTM Metrics
- Lead to onboarding conversion.
- Week 1 activation rate.
- Week 4 retention rate.
- Cost per onboarded restaurant (if ad spend used).

## Expected Outputs
- Best-performing acquisition channel.
- Most convincing value message.
- Early pricing guidance for next phase.
