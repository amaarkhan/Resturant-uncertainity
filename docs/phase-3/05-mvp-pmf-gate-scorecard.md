# MVP to PMF Gate Scorecard

## Purpose
Use this scorecard at pilot end to decide PMF direction from MVP evidence.

## Input Window
- Baseline period: 10-14 days
- MVP period: 4-6 weeks

## Metrics
1. Waste reduction percent (target >= 10)
2. Stockout reduction percent (target >= 15)
3. Recommendation follow rate (target >= 60)
4. Confidence/trust score average (target >= 4.0)
5. Week 4 active usage (target >= 40)
6. Willingness-to-pay signal (target >= 20)

## Decision Rule
- PMF Direction Positive: 4 or more targets met including waste reduction.
- Promising Not Yet PMF: 2-3 targets met, clear trend positive.
- Pivot Needed: 0-1 targets met or trust/retention weak.
